Pushes to the stack memory, the 16-bit source.

Stops the system clock and enters HALT mode.

Although the system clock is stopped in this state, the oscillator circuit and LCD controller continue to operate.

fn get_address(&mut self, addr: At) -> u16

Returns the 16-bit data stored at address represented by addr.
At can be the 16-bit register HL, BC, or DE,
a 16-bit immediate data (Imm16) or
an 8-bit data (Imm8 or 8-bit register C).
When addr represents an 8-bit data, the returned value is obtained by setting the most significant byte to 0xFF and the least significant to the 8-bit data to form an address in the range 0xFF00-0xFFFF.

fn get_operand8(&mut self, operand: Operand8) -> u8

Returns the 8-bit data represented by operand.

an 8-bit immediate data (Imm8) or
an 8-bit data stored at location (Addr(at) where at represents the location).

source

fn get_operand16(&mut self, operand: Operand16) -> u16

Returns the 16-bit data represented by operand.
Operand16 is either a 16-bit register (AF, BC, DE, HL, SP), a 16-bit or 8-bit immediate data (Imm16 or Imm8) or, the 16-bit data stored at location (Addr(at) where at represents the location). When operand represents an 8-bit data, the returned value is obtained by setting the most significant byte to 0x00 and the least significant byte to the 8-bit data to form a 16-bit value in the range 0x0000-0x00FF.
